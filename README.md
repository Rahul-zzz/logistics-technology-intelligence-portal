# 🚚 Logistics Technology Intelligence Portal

A Streamlit-based analytics dashboard designed to evaluate logistics companies, technology ecosystems, AI readiness, and operational maturity.

## 📌 Overview

The Logistics Technology Intelligence Portal provides a centralized platform to:

- Analyze logistics companies and their technology stack
- Explore ERP, TMS, and Fleet Management systems
- Evaluate AI readiness and technology maturity
- Compare companies side-by-side
- Generate strategic insights and recommendations

---

## 🎯 Objectives

- Understand technology adoption in logistics organizations
- Benchmark companies based on digital maturity
- Assess AI readiness levels
- Identify technology gaps and improvement opportunities
- Support data-driven business decisions

---

## ✨ Features

### 🏢 Company Explorer
- Company profile overview
- Technology stack analysis
- Top customers and competitors
- Operational challenges and opportunities
- Associated systems mapping

### ⚙️ System Explorer
- ERP, TMS, and Fleet system analysis
- Features, advantages, and limitations
- Vendor and category information
- Companies using each system

### 📊 Technology Dashboard
- Technology maturity analysis
- AI readiness comparison
- Company size distribution
- Most-used systems analysis
- System category breakdown

### 🤖 AI Readiness Dashboard
- AI readiness ranking
- AI leaders identification
- Improvement target analysis
- AI opportunity assessment
- Strategic AI recommendations

### ⚖️ Company Comparison
- Side-by-side company benchmarking
- Technology maturity comparison
- AI readiness comparison
- Detailed attribute comparison

### 📈 Insights & Recommendations
- Executive summary
- Industry technology gaps
- Strategic recommendations
- High-impact technology analysis

## 🚀 Installation

### Clone Repository

```bash
git clone <repository-url>
cd logistics-technology-intelligence-portal
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
